# Чай с лимоном

## <mark style="color:purple;">Чай с лимоном</mark>

Чайник обдайте кипятком, положите в него заварку, залейте и дайте постоять несколько минут, затем добавьте сок лимона.

| Продукты (на 1 порцию) |   |          |
| ---------------------- | - | -------- |
| чай                    | 2 | ч. ложки |
| сок лимонный           |   | по вкусу |
